## 🎯 立即开始（3步）

### 1️⃣ 获取代码
```bash
cd ~/clawd/products/auto-reply-pro
```

### 2️⃣ 配置环境
```bash
# 复制配置模板
cp .env.production .env

# 编辑配置（必需）
nano .env

# 设置 API_KEY 和 DEEPSEEK_API_KEY
```

### 3️⃣ 启动服务
```bash
# 方式1: 交互式部署
./scripts/deploy.sh

# 方式2: 直接启动
npm start

# 方式3: Docker
docker-compose up -d
```
