## 🚀 快速开始

### 1. 部署（3分钟）

```bash
# 克隆项目
git clone https://github.com/your-repo/auto-reply-pro.git
cd auto-reply-pro

# 运行部署脚本
chmod +x scripts/deploy.sh
./scripts/deploy.sh
```

### 2. 配置（2分钟）

```bash
# 编辑配置文件
nano .env

# 必需配置
API_KEY=your-api-key
DEEPSEEK_API_KEY=your-deepseek-key
```

### 3. 启动（1分钟）

```bash
# 方式1: 直接启动
node src/index-secure.js

# 方式2: PM2（推荐）
pm2 start src/index-secure.js --name auto-reply-pro

# 方式3: Docker
docker-compose up -d
```

### 4. 访问

- **Web Chat**: http://localhost:3002/api/chat
- **管理面板**: http://localhost:3003
