# 🎯 Auto-Reply Pro - 消息发送器使用指南

## 📋 功能说明

**这是什么？**
一个 Web 界面的消息发送工具，你可以：
1. 在网页上输入消息
2. 点击"发送"按钮
3. Auto-Reply Pro 帮你发送到飞书

**使用场景**：
- 快速发送消息给客户
- 批量发送通知
- 测试消息发送功能
- 演示产品能力

---

## 🚀 使用步骤

### 1️⃣ 打开 Web 界面

```
http://localhost:3003
```

### 2️⃣ 填写信息

- **平台**：选择"飞书"
- **接收者 ID**：输入飞书用户 ID（例如：ou_d5695b3a7239c8fb1a3af9a46140b27e）
- **消息内容**：输入你想发送的消息（例如："hello"）

### 3️⃣ 点击发送

点击"📤 发送消息"按钮

### 4️⃣ 查看结果

- ✅ 成功：显示"消息发送成功"
- ❌ 失败：显示错误信息

## ✅ 特性

- ✅ **简单易用**：填写表单，点击发送
