## 🚀 快速开始

### 1. 安装依赖
```bash
cd ~/clawd/products/auto-reply-pro
npm install
```

### 2. 配置环境变量
```bash
# .env
DEEPSEEK_API_KEY=your_key
FEISHU_APP_ID=your_id
FEISHU_APP_SECRET=your_secret
```

### 3. 启动服务
```bash
npm start
```

### 4. 运行演示
```bash
node demo-proactive-messaging.js
```
